Array
	Java's array
	Java语言是经典的静态语言，所以Java的数据也是静态的，因此
Java的数组初始化后，其长度就是不可变的。Java中有俩种数组初始化
的方式：
	- 静态初始化 （代码中指定每个元素的初始值，JVM决定其长度）
	- 动态初始化 （代码指定数组的长度，JVM为每个元素进行初始化）
	
	DefineAndInit	演示数组的定义以及初始化
	DamoOne			演示数组的使用。（排序）

OOP
	Java面向对象的思想。类的定义封装,继承,重载,多态的编程思想
	类和对象  类是对象的抽象，对象是类的实例 
	构造方法： 访问属性  类名(参数类型 参数...)
	类的属性访问：  object.property
	类的方法访问：  object.function
	
	final关键字
		-final修饰的类不能有子类，例如String
		-final修饰的方法不能被子类覆写
		-final修饰的变量及成常量 不能被修改
	类继承：
		class parentClass{}
		class childClass extends parentClass()
		子类实例化时先调用父类的构造，再调自己的构造
	覆写
		在继承的关系中发送一种覆写关系。所谓覆写就是指子类定义与
		父类同名的方法。
		当方法被覆写后，子类对象调用的被覆写后的方法
	
 抽象类：
 	有一个方法是抽象方法的类及成抽象类
 	抽象类和抽象方法必须有abstract关键字修饰
 	抽象方法只需声明，不需实现
 	子类继承抽象类并实现所有方法才能实例化对象，否则不能用于实例化对象
接口：
	接口是Java中一个相当重要的概念。接口中可以定义方法和全局常量。其定义格式如下
	interface name{
		global variable;
		public fun();
	}
	注意接口中的方法不论写不写public都是public权限的。
	子类可以通过implements继承接口并实现相应的方法
	
Java只支持单继承。但可以通过继承一个基类并实现多个接口的方式可以完成一些多继承的特性

包及访问权限
类的成员访问控制符有四种分配是public, protected, default, private
Private如果一个成员方法或成员变量名之前使用private访问控制符，那么这个成员只能在这个类的内部使用
Protected如果一个成员方法或成员变量名前使用了protected访问控制符，那么这个成员即可以被同包中的其他类访问，也可以被不同包中的子类访问。
Default如果一个成员方法或成员变量名前没有使用任何访问控制符，就称为默认访问控制（default）或是友元（friendly）或是包类型的package。对于默认访问控制成员
可以被这个包中其他类访问。如果一个子类与父类位于不同的包中。子类也不能访问父类的默认访问控制的成员。
Public如果一个成员方法或成员变量名前使用了public访问控制符，那么这个成员即可以被所有的类访问。不关访问类与被访问类是否在同一个包中。

ThreadDemo
	Java中的多线程
	Java中实现多线程程序有俩种方式
		* 继承Thread类	(ThreadExtends 所演示的方式)
		* 实现Runnable接口 (ThreadByRunnable所演示的方式)
	不论以然后方式实现多线程，都必须以Thread.start()方法启动一个线程，不能直接调用run方法。
	另外以继承Thread类创建的线程对象不能调用多次start
