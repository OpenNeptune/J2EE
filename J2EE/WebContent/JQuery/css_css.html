·css(name|pro|[,val|fn])
	访问/设置匹配元素的样式属性。
	name
		要访问的属性名称
	properties
		要设置为样式属性的名/值对
	name,valueString
		属性名,属性值
	name,function(index, value)
		1:属性名
		2:此函数返回要设置的属性值。
		接受两个参数，index为元素在对象集合中的索引位置，value是原先的属性值。
